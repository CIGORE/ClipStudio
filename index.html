Panduan Penggunaan ClipStudio

ClipStudio dirancang untuk membantu Anda menemukan momen paling viral dari video YouTube dan mengidentifikasi konten trending di berbagai penjuru dunia. Gunakan Clip Highlighter untuk menganalisis video secara mendalam, atau Viral Finder untuk menemukan konten populer berdasarkan niche dan region pilihan Anda.

🚀 Cara Menggunakan ClipStudio
01

Siapkan API Key

Butuh YouTube API Key (wajib) dan AI API Key pilihan. Semua key hanya tersimpan di browser Anda.

🎯 Rekomendasi pemula: Gunakan Google Gemini — gratis, tidak ada masalah CORS, tidak perlu kartu kredit.

① YouTube Data API v3 (Wajib)

  • Buka console.cloud.google.com
  • Buat project → Enable YouTube Data API v3
  • Credentials → Create API Key → Copy (format: AIzaSy…)
⚠️ Kuota gratis: 10.000 unit/hari. Tiap pencarian ~100 unit.

② Google Gemini — GRATIS & Direkomendasikan

  • Buka aistudio.google.com
  • Login → klik Get API Key → Create API Key
  • Format sama: AIzaSy…
✓ Gratis 1.500 req/hari ✓ Tanpa kartu kredit ✓ Tidak ada masalah CORS
❌ Claude & OpenAI memblokir request langsung dari browser (CORS policy). Untuk menggunakannya, Anda perlu backend/proxy sendiri.
⚠️ Jangan bagikan API key Anda kepada siapa pun.
02

Gunakan Clip Highlighter

Masukkan URL YouTube, atur opsi clip, klik Analyze. AI menganalisis video dan menemukan segmen paling viral dengan skor, alasan, dan caption siap pakai.

✓ Deteksi Hook ✓ Viral Score ✓ AI Caption ✓ Timeline Visual
03

Gunakan Viral Finder

Masukkan keyword, pilih negara target, klik Find Viral. Sistem mencari video viral real-time dengan brand safety score.

✓ Prioritaskan US, UK, Australia untuk CPM tertinggi.
⚠️ Max 3 region sekaligus untuk hemat kuota API.
04

Export dan Monetisasi

Export JSON/CSV, gunakan timestamp untuk memotong di CapCut/Premiere, salin caption ke TikTok/Reels/Shorts.

❌ Selalu tambahkan nilai orisinal — jangan upload ulang tanpa modifikasi.
